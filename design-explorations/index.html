BabyBedtimeTeller — three directions

Pick one to iterate on. Each is a self-contained static page.

Option A
Papercut Storybook
Dusk plum sky & cream paper. A literal open-book spread with a die-cut moon window holding the form. Fraunces + Nunito. Warm, whimsical, tactile.
Option B
Hushed Observatory
Near-black with pinprick stars and a single glowing ember accent. A nightlight-orb CTA is the only warmth on the page. Newsreader italic + Inter + mono captions. Spare, editorial, quiet.
Option C
Stitched Nursery
Daylight linen, sage & blush patchwork quilt with dashed stitch borders and a hand-stitched thread trail connecting each patch. Bitter slab + Karla + Caveat tags. Cozy, tactile, handmade.
